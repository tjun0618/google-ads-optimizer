## 1. 核心公式

### 盈亏平衡CPC

```
盈亏平衡CPC = 产品售价 × 佣金率 × Amazon预估转化率
安全CPC上限 = 盈亏平衡CPC × 安全系数
```

### 品牌级EPC（如有YP数据）

```
品牌EPC = YP Commission ÷ YP Clicks
品牌ROI = (YP Commission - Google Ads Cost) / Google Ads Cost
```

**注意**：YP Clicks包含所有渠道，可能大于Google Ads Clicks。关键词级EPC无法计算（YP不区分关键词）。

### 安全系数建议

| 产品类型 | 佣金范围 | 安全系数 |
|---------|---------|---------|
| 高利润产品 | 佣金>$50 | 0.6-0.7 |
| 中利润产品 | 佣金$20-$50 | 0.4-0.5 |
| 低利润产品 | 佣金<$20 | 0.3-0.4 |

## 2. Amazon预估转化率参考表

| 品类 | 预估转化率 | 说明 |
|------|-----------|------|
| 低卷入快消品（零食、日用品） | 8-12% | 价格低，决策快 |
| 中卷入标准品（宠物用品、清洁工具） | 5-8% | 需简单比较 |
| 高卷入耐用品（吹风机、电器） | 3-5% | 价格高，需对比 |
| 健康/保健品 | 2-4% | YMYL品类，更谨慎 |
| 极高价/专业设备 | 1-3% | 决策周期长 |

**调整因子**：
- Review数量 < 50：× 0.7
- 评分 < 4.0：× 0.6
- 有A+ Content和视频：× 1.2
- Prime eligible：× 1.1
- Currently unavailable：= 0

## 3. 关键词利润评级

| 评级 | 条件 | 颜色 | 行动 |
|------|------|------|------|
| **A - 高利润** | CPC < 安全上限 × 0.5 | 绿 | 加码：提高出价10-20% |
| **B - 盈利** | 安全上限 × 0.5 ≤ CPC < 安全上限 | 浅绿 | 维持：保持不变 |
| **C - 边缘** | 安全上限 ≤ CPC < 盈亏平衡CPC | 黄 | 警惕：降低出价 |
| **D - 亏损** | CPC ≥ 盈亏平衡CPC | 红 | 暂停：立即暂停 |
| **? - 未知** | 无点击数据 | 灰 | 观察：积累数据 |
