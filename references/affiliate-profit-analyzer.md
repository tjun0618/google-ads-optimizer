> 专为Amazon联盟营销客设计的关键词级利润分析模块
> 核心目标：计算每个关键词的盈亏状态，识别亏损词和高利润机会

## 1. 核心公式

### 1.1 盈亏平衡CPC（Breakeven CPC）

```
盈亏平衡CPC = 产品售价 × 佣金率 × Amazon预估转化率

或

盈亏平衡CPC = 预估每单佣金 ÷ 所需点击数
            = (产品售价 × 佣金率) ÷ (1 ÷ Amazon预估转化率)
            = 产品售价 × 佣金率 × Amazon预估转化率
```

**示例**：
- 产品：SRI DryQ Hair Dryer | $339.99 | 佣金率22.5%
- 预估Amazon转化率：5%（点击到购买）
- 每单佣金：$339.99 × 22.5% = $76.50
- 盈亏平衡CPC：$339.99 × 22.5% × 5% = $3.83
- 即：只要CPC < $3.83，理论上每个点击都是盈利的

### 1.2 实际EPC（Earnings Per Click）

如有Amazon Associates订单报告：

```
实际EPC = 总佣金收入 ÷ 总Google Ads点击数
```

**解读**：
- EPC > 盈亏平衡CPC → 盈利
- EPC < 盈亏平衡CPC → 亏损
- EPC ≈ 盈亏平衡CPC × 0.3-0.5 → 常见（Amazon转化率通常低于预期）

### 1.3 安全CPC上限（Safe CPC Cap）

Affiliate客的实际转化率通常低于理论值，建议设置保守上限：

```
安全CPC上限 = 盈亏平衡CPC × 安全系数

安全系数建议：
- 高利润产品（佣金>$50）：0.6-0.7
- 中利润产品（佣金$20-$50）：0.4-0.5
- 低利润产品（佣金<$20）：0.3-0.4
```

**为什么需要安全系数？**
- Amazon Associates的24小时Cookie窗口内，用户可能购买其他低佣金产品
- 退货会扣除佣金
- Google Ads点击 ≠ Amazon Associates点击（部分用户浏览器拦截、JavaScript禁用）
- 预估转化率往往过于乐观

## 2. Amazon预估转化率参考表

不同品类的Amazon预估转化率（点击到购买）：

| 品类 | 预估转化率 | 说明 |
|------|-----------|------|
| 低卷入快消品（零食、日用品） | 8-12% | 价格低，决策快，冲动购买 |
| 中卷入标准品（宠物用品、清洁工具） | 5-8% | 需要简单比较，但价格不高 |
| 高卷入耐用品（吹风机、电器） | 3-5% | 价格高，需要看评论、对比 |
| 健康/保健品 | 2-4% | YMYL品类，用户更谨慎 |
| 极高价/专业设备 | 1-3% | 决策周期长，多次访问才购买 |

**调整因子**：
- Review数量 < 50：转化率 × 0.7
- 评分 < 4.0：转化率 × 0.6
- 有A+ Content和视频：转化率 × 1.2
- Prime eligible：转化率 × 1.1
- Currently unavailable：转化率 = 0

## 3. 关键词利润评级

对每个广告组/关键词进行利润评级：

| 评级 | 条件 | 颜色 | 行动 |
|------|------|------|------|
| **A - 高利润** | CPC < 安全上限 × 0.5 | 绿色 | 加码：提高出价10-20%，考虑扩展匹配 |
| **B - 盈利** | 安全上限 × 0.5 ≤ CPC < 安全上限 | 浅绿 | 维持：监控表现，保持不变 |
| **C - 边缘** | 安全上限 ≤ CPC < 盈亏平衡CPC | 黄色 | 警惕：降低出价或收紧匹配 |
| **D - 亏损** | CPC ≥ 盈亏平衡CPC | 红色 | 暂停：立即暂停或大幅降低出价 |
| **? - 未知** | 无点击数据 | 灰色 | 观察：保持运行积累数据 |

### Step 4.8: 预估转化率校准

```

如果实际转化率 < 预估转化率 × 0.5：
  → 检查产品页面（评论、评分、库存、价格）

  → 产品表现超预期，可提高CPC上限加码投放
```
