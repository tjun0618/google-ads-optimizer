# YP Affiliate Checklist — Google Ads Optimization Skill v3.0

> YP平台联盟营销客的投放检查清单
>
> **v3.0定位**：本文件提供优化技能在Step 6.8（政策合规与库存检查）时使用的检查项。
> 更全面的止损决策框架、Cookie策略、选品评级已迁移到诊断技能。

---

## 1. 政策合规检查（Step 6.8使用）

### YP平台合规
- [ ] YP联盟链接正确（Deep Link指向正确产品页，追踪参数完整）
- [ ] 广告中价格与Amazon一致（或标注"Price may vary"）
- [ ] 无 incentivize 点击措辞（"点击我的链接获得折扣"等）
- [ ] 确认品牌允许通过YP平台进行联盟推广

### Google Ads政策合规
- [ ] 落地页为Amazon实际产品页（非Bridge Page）
- [ ] 广告文案无夸大声明（特别是健康/保健品类）
- [ ] 无虚假折扣/优惠声明
- [ ] 未隐瞒最终落地页（重定向透明）

### Amazon产品页面检查
- [ ] 产品可用（Currently available）
- [ ] Buy Box由Amazon自营或FBA卖家持有
- [ ] 评分 ≥ 4.0（如<4.0则转化率×0.6）
- [ ] Review数量 ≥ 50（如<50则转化率×0.7）
- [ ] 30天内价格波动 < 20%
- [ ] 近期无大量负面评价（30天内负面<10%）

---

## 2. 持续监控清单

### 每日（5分钟）
- [ ] Amazon产品页状态（可用/不可用）
- [ ] Amazon价格变动（>10%需重算盈亏平衡CPC）
- [ ] Google Ads Campaign状态（是否被暂停/拒登）
- [ ] 前一日费用 vs 预算（是否超支）

### 每周（30分钟）
- [ ] 下载搜索字词报告，审查"都没有"状态的字词
- [ ] 添加新否定词
- [ ] 检查产品Review新增情况
- [ ] 检查库存状态（特别是FBA库存）

### 每两周（1小时）
- [ ] 关键词利润评级复核
- [ ] 评估Campaign整体ROI
- [ ] 决定继续/止损/加码

---

## 3. 止损信号速查

| 信号 | 严重程度 | 行动 |
|------|---------|------|
| Amazon产品变为Currently unavailable | 致命 | 立即暂停所有Campaign |
| 产品评分从4.5降至3.5以下 | 致命 | 暂停，等待评分回升 |
| 实际EPC连续2周为0 | 严重 | 深度诊断→如无误→止损换品 |
| 平均CPC超过盈亏平衡CPC | 严重 | 暂停亏损词 |
| Amazon佣金率被削减50%以上 | 严重 | 重新计算盈亏平衡 |

**止损线计算**（来自诊断报告）：
- 止损点击数 = 盈亏平衡CPC对应所需点击数 × 3
- 止损费用 = 止损点击数 × 当前平均CPC
