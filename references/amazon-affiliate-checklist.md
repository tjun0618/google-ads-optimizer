# YP Affiliate Checklist — Google Ads Optimization Skill

> YP平台联盟营销客的投放前检查清单与持续监控指南
> 核心目标：确保产品可盈利、库存可用、政策合规，避免投放后才发现问题

---

## 1. 投放前检查清单（Pre-Launch Checklist）

在启动任何新的Google Ads Campaign前，必须完成以下检查：

### 1.1 产品利润 viability 检查

| 检查项 | 标准 | 如果不达标 | 优先级 |
|--------|------|-----------|--------|
| 佣金率 ≥ 10% | 是/否 | 佣金太低，难以覆盖CPC | P0 |
| 产品售价 ≥ $20 | 是/否 | 低客单价产品CPC必须极低 | P0 |
| 盈亏平衡CPC ≥ $0.30 | 是/否 | CPC低于$0.30极难实现 | P0 |
| 预估每单佣金 ≥ $5 | 是/否 | 每单佣金太低不划算 | P1 |
| 竞品CPC可接受 | 搜索验证 | 如竞品品牌词CPC>$5，慎投 | P1 |

**快速利润计算**：
```
1. 产品售价 × 佣金率 = 每单佣金
2. 每单佣金 × 预估Amazon转化率(3-8%) = 盈亏平衡CPC
3. 盈亏平衡CPC × 0.6 = 安全CPC上限
4. 如果安全CPC上限 < $0.30 → 不建议投放
```

### 1.2 Amazon产品页面检查

| 检查项 | 标准 | 如果不达标 | 行动 |
|--------|------|-----------|------|
| 产品可用 | Currently available | 不可用 | 不投放 |
| Buy Box | Amazon自营或FBA卖家持有 | 第三方卖家Buy Box | 风险高，慎投 |
| Prime配送 | Prime eligible | 非Prime | 转化率降低30-50% |
| 评分 ≥ 4.0 | 是/否 | < 4.0 | 转化率×0.6，收紧CPC |
| Review数量 ≥ 50 | 是/否 | < 50 | 转化率×0.7，收紧CPC |
| A+ Content | 有/无 | 无 | 转化率略低，可接受 |
| 产品视频 | 有/无 | 无 | 转化率略低，可接受 |
| 近期负面评价 | < 10% of 30天内评论 | > 10% | 风险信号，观察 |
| 价格稳定 | 30天内价格波动 < 20% | 波动大 | 佣金不稳定 |

### 1.3 竞争环境检查

| 检查项 | 方法 | 标准 |
|--------|------|------|
| Google搜索竞品广告 | 搜索核心关键词 | 如首页全是Amazon/Sephora/品牌官网广告，竞争极激烈 |
| SEMrush/SpyFu竞品分析 | 工具查询 | 如竞品月广告支出>$10K，CPC可能很高 |
| Amazon竞品数量 | Amazon搜索 | 如首页>20个竞品且价格更低，转化困难 |

### 1.4 Google Ads政策合规检查

**Affiliate链接投放的特殊政策风险**：

| 检查项 | 要求 | 违规后果 |
|--------|------|---------|
| 落地页质量 | 不能是Bridge Page（仅用于跳转的页面） | 广告被拒登 |
| 落地页内容 | 必须有实质性原创内容，不能纯复制Amazon描述 | 广告被拒登 |
| 广告文案诚实 | 不能夸大产品效果（特别是健康品类） | 广告被拒登 |
| 折扣/优惠声明 | 必须有实际优惠，不能虚假 | 广告被拒登 |
| 品牌词使用 | 可以使用品牌词（Google允许） | — |
| 重定向 | 不能隐瞒最终落地页 | 账户被封 |

**直接投放Amazon联盟链接的特殊注意**：
- Google Ads允许直接投放Amazon链接（与Bing Ads不同）
- 但落地页必须是Amazon.com上的实际产品页
- 不能使用中间跳转页面（如yourdomain.com/redirect?asin=xxx）
- 广告文案必须与Amazon页面一致

### 1.5 YP平台合规检查

| 检查项 | 要求 |
|--------|------|
| YP联盟链接正确 | 确保Deep Link指向正确的产品页，追踪参数完整 |
| 价格标注准确 | 广告中显示的价格必须与Amazon一致（或标注"Price may vary"） |
| 禁止 incentivize | 不能用"点击我的链接获得折扣"等诱导点击 |
| 品牌授权 | 确认该品牌允许通过YP平台进行联盟推广 |

## 2. 持续监控清单（Ongoing Monitoring）

### 2.1 每日检查（5分钟）

```markdown
□ Amazon产品页面状态（可用/不可用）
□ Amazon价格变动（是否有大幅降价/涨价）
□ Google Ads Campaign状态（是否被暂停/拒登）
□ 前一日费用 vs 预算（是否超支）
```

### 2.2 每周检查（30分钟）

```markdown
□ 下载搜索字词报告，审查"都没有"状态的字词
□ 添加新的否定词（从搜索字词中提取无关词）
□ 检查Amazon Associates订单报告
□ 计算实际EPC vs 盈亏平衡CPC
□ 检查产品Review新增情况（是否有大量负面评价）
□ 检查竞品价格变动
□ 检查库存状态（特别是FBA库存）
```

### 2.3 每两周检查（1小时）

```markdown
□ 完整的关键词利润评级复核
□ 审查暂停的关键词是否有恢复价值
□ 检查是否有新的高利润搜索词出现
□ 评估Campaign整体ROI（实际佣金 vs Google Ads费用）
□ 决定继续/止损/加码（使用报告中的决策框架）
□ 检查Amazon Associates佣金率是否有调整
```

## 3. Amazon Associates Cookie窗口策略

### 3.1 24小时Cookie机制

```
用户点击你的联盟链接 → 24小时Cookie激活 → 24小时内购买的任何商品你都赚佣金

关键洞察：
- 用户点了SRI Dryer广告 → 买了猫粮 → 你赚猫粮佣金
- 用户点了SRI Dryer广告 → 没买 → 24小时后Cookie失效 → 你再也不会赚到这笔佣金
```

### 3.2 Cookie窗口策略

**策略1: 高流量品类截流**
- 投放高搜索量但低佣金的产品的广告
- 目的不是赚这个产品的佣金，而是获得24小时Cookie窗口
- 用户在这24小时内购买其他高佣金产品，你赚高佣金
- **风险**：Google Ads费用 > 附带订单佣金 = 亏损
- **适用**：只有当你有数据证明附带订单率>30%时才使用

**策略2: 促进即时购买**
- 广告文案强调"Limited stock"、"Prime delivery tomorrow"等促进即时决策的信息
- 减少用户"先收藏，以后再看"的行为
- 因为"以后再看"= Cookie过期 = 你赚不到

**策略3: 高佣金产品优先**
- 优先投放佣金率>15%的产品
- 即使点击量相同，高佣金产品的容错空间更大
- 低佣金产品（如书籍4%）需要极低的CPC才能盈利

## 4. 库存/价格监控自动化建议

### 4.1 手动监控流程

```markdown
每日:
1. 打开Amazon产品页
2. 确认"Add to Cart"按钮可用（非"Currently unavailable"）
3. 记录当前价格
4. 如价格变化>10%，重新计算盈亏平衡CPC
5. 如不可用，立即暂停所有Campaign

每周:
1. 检查Amazon Associates后台Orders报告
2. 记录订单数、商品、佣金
3. 计算实际EPC
4. 与预估对比，校准模型
```

### 4.2 半自动化监控（推荐）

使用工具或脚本：
- **Keepa** (keepa.com)：监控Amazon价格历史和库存状态
- **CamelCamelCamel**：价格追踪和提醒
- **自制脚本**：用Python + BeautifulSoup定期检查Amazon页面状态

**监控脚本逻辑**：
```python
# 伪代码
def check_product(asin):
    page = fetch_amazon_page(asin)
    availability = extract_availability(page)
    price = extract_price(page)
    rating = extract_rating(page)
    review_count = extract_review_count(page)
    
    if availability != "In Stock":
        alert(f"产品{asin}不可用，请暂停Campaign")
        return "STOP"
    
    if price_changed > 10%:
        alert(f"产品{asin}价格变化{price_changed}%，请重新计算盈亏平衡CPC")
        return "RECALCULATE"
    
    if rating < 4.0 or review_count < 50:
        alert(f"产品{asin}评分/评论下降，转化率可能降低")
        return "WARNING"
    
    return "OK"
```

## 5. 止损决策框架

### 5.1 阶段性决策节点

```markdown
### 投放1周后
- 检查：是否有Amazon订单？
- 有订单 → 计算实际EPC，继续优化
- 无订单 → 检查Amazon页面状态，确认无误后继续观察

### 投放2周后
- 检查：实际EPC vs 盈亏平衡CPC
- 实际EPC > 盈亏平衡 × 0.5 → 继续，加码高利润Campaign
- 实际EPC > 0 但 < 盈亏平衡 × 0.3 → 维持，收紧CPC
- 实际EPC = 0 → 进入深度诊断

### 投放3-4周后
- 检查：累计投入 vs 累计佣金
- 累计佣金 > 累计投入 × 0.5 → 继续
- 累计佣金 < 累计投入 × 0.2 → 考虑止损
- 零佣金且投入 > ¥500 → 强烈建议止损换品

### 投放1个月后
- 全面评估：实际EPC、CPC趋势、订单趋势
- 决定：继续/加码/维持/止损/换品
```

### 5.2 止损信号清单

出现以下任一信号，考虑立即止损：

| 信号 | 严重程度 | 行动 |
|------|---------|------|
| Amazon产品变为Currently unavailable | 致命 | 立即暂停所有Campaign |
| 产品评分从4.5降至3.5以下 | 致命 | 暂停，等待评分回升 |
| 实际EPC连续2周为0 | 严重 | 检查页面→如无误→止损换品 |
| 平均CPC超过盈亏平衡CPC | 严重 | 暂停亏损词，整体评估 |
| Amazon佣金率被削减50%以上 | 严重 | 重新计算盈亏平衡，可能止损 |
| 竞品大幅降价（>30%） | 中等 | 监控转化率，如下降则收紧预算 |
| Google Ads Campaign连续被拒登 | 中等 | 解决政策问题，否则止损 |

## 6. 选品利润评级框架

在决定投放哪个产品前，用此框架快速评级：

```markdown
| 评级 | 条件 | 预期结果 | 投放建议 |
|------|------|---------|---------|
| **S级** | 佣金>$50, 评分>4.3, Review>200, 评分稳定 | 高利润，可持续 | 优先投放，加大预算 |
| **A级** | 佣金$20-$50, 评分>4.0, Review>100 | 稳定盈利 | 正常投放 |
| **B级** | 佣金$10-$20, 评分>3.8, Review>50 | 微利，需严格控制CPC | 谨慎投放，测试为主 |
| **C级** | 佣金$5-$10, 或评分3.5-3.8 | 盈亏边缘 | 仅投Brand词，低预算测试 |
| **D级** | 佣金<$5, 或评分<3.5, 或Review<20 | 大概率亏损 | 不建议投放 |

加分项：
- Prime eligible (+1级)
- 有A+ Content (+0.5级)
- 有视频 (+0.5级)
- 季节性需求高峰 (+1级)

减分项：
- 近期有召回/负面新闻 (-1级)
- 价格频繁波动 (-0.5级)
- 大量1-2星评论 (-0.5级)
```

## 7. 快速参考卡片

**盈亏平衡CPC速查表**（基于5% Amazon转化率）：

| 售价 | 佣金率 | 每单佣金 | 盈亏平衡CPC | 安全上限(×0.6) |
|------|--------|---------|------------|---------------|
| $30 | 10% | $3.00 | $0.15 | $0.09 |
| $30 | 15% | $4.50 | $0.23 | $0.14 |
| $50 | 10% | $5.00 | $0.25 | $0.15 |
| $50 | 15% | $7.50 | $0.38 | $0.23 |
| $100 | 10% | $10.00 | $0.50 | $0.30 |
| $100 | 15% | $15.00 | $0.75 | $0.45 |
| $200 | 10% | $20.00 | $1.00 | $0.60 |
| $200 | 15% | $30.00 | $1.50 | $0.90 |
| $300 | 10% | $30.00 | $1.50 | $0.90 |
| $300 | 20% | $60.00 | $3.00 | $1.80 |
| $500 | 10% | $50.00 | $2.50 | $1.50 |
| $500 | 20% | $100.00 | $5.00 | $3.00 |

**结论**：
- 低客单价($30-50) + 低佣金率(<15%) = 几乎不可能盈利（CPC上限<$0.30）
- 高客单价($200+) + 高佣金率(>15%) = 有充足利润空间
- **选品时优先选择：售价>$100 且 佣金率>10% 的产品**
