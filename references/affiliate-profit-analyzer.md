# Affiliate Profit Analyzer — Google Ads Optimization Skill

> 专为Amazon联盟营销客（YP平台）设计的利润分析模块
> 核心目标：计算品牌级实际ROI，预估关键词级盈亏状态，识别亏损词和高利润机会
> ⚠️ 归因限制：YP平台只提供品牌级聚合数据，实际利润分析以品牌级为准，关键词级为预估

---

## 1. 核心公式

### 1.1 盈亏平衡CPC（Breakeven CPC）

```
盈亏平衡CPC = 产品售价 × 佣金率 × Amazon预估转化率

或

盈亏平衡CPC = 预估每单佣金 ÷ 所需点击数
            = (产品售价 × 佣金率) ÷ (1 ÷ Amazon预估转化率)
            = 产品售价 × 佣金率 × Amazon预估转化率
```

**示例**：
- 产品：SRI DryQ Hair Dryer | $339.99 | 佣金率22.5%
- 预估Amazon转化率：5%（点击到购买）
- 每单佣金：$339.99 × 22.5% = $76.50
- 盈亏平衡CPC：$339.99 × 22.5% × 5% = $3.83
- 即：只要CPC < $3.83，理论上每个点击都是盈利的

### 1.2 品牌级实际EPC（Earnings Per Click）

如有YP平台品牌转化报表：

```
品牌实际EPC = YP品牌总Commission ÷ YP品牌总Clicks

品牌实际ROI = (YP品牌总Commission - Google Ads品牌总Cost) / Google Ads品牌总Cost
```

**解读**：
- 品牌EPC > 盈亏平衡CPC → 品牌整体盈利
- 品牌EPC < 盈亏平衡CPC → 品牌整体亏损
- 注意：YP Clicks通常包含所有渠道，可能大于Google Ads Clicks

**关键词级EPC无法计算**：YP数据不区分关键词，只能基于CPC vs 盈亏平衡做**预估**

### 1.3 安全CPC上限（Safe CPC Cap）

Affiliate客的实际转化率通常低于理论值，建议设置保守上限：

```
安全CPC上限 = 盈亏平衡CPC × 安全系数

安全系数建议：
- 高利润产品（佣金>$50）：0.6-0.7
- 中利润产品（佣金$20-$50）：0.4-0.5
- 低利润产品（佣金<$20）：0.3-0.4
```

**为什么需要安全系数？**
- YP平台佣金可能有退货扣除、订单取消等调整
- Google Ads点击 ≠ YP追踪点击（部分用户浏览器拦截、跳转丢失）
- YP Clicks是所有渠道总和，Google Ads只是其中一个子集
- 预估转化率往往过于乐观

## 2. Amazon预估转化率参考表

不同品类的Amazon预估转化率（点击到购买）：

| 品类 | 预估转化率 | 说明 |
|------|-----------|------|
| 低卷入快消品（零食、日用品） | 8-12% | 价格低，决策快，冲动购买 |
| 中卷入标准品（宠物用品、清洁工具） | 5-8% | 需要简单比较，但价格不高 |
| 高卷入耐用品（吹风机、电器） | 3-5% | 价格高，需要看评论、对比 |
| 健康/保健品 | 2-4% | YMYL品类，用户更谨慎 |
| 极高价/专业设备 | 1-3% | 决策周期长，多次访问才购买 |

**调整因子**：
- Review数量 < 50：转化率 × 0.7
- 评分 < 4.0：转化率 × 0.6
- 有A+ Content和视频：转化率 × 1.2
- Prime eligible：转化率 × 1.1
- Currently unavailable：转化率 = 0

## 3. 关键词利润评级

对每个广告组/关键词进行利润评级：

| 评级 | 条件 | 颜色 | 行动 |
|------|------|------|------|
| **A - 高利润** | CPC < 安全上限 × 0.5 | 绿色 | 加码：提高出价10-20%，考虑扩展匹配 |
| **B - 盈利** | 安全上限 × 0.5 ≤ CPC < 安全上限 | 浅绿 | 维持：监控表现，保持不变 |
| **C - 边缘** | 安全上限 ≤ CPC < 盈亏平衡CPC | 黄色 | 警惕：降低出价或收紧匹配 |
| **D - 亏损** | CPC ≥ 盈亏平衡CPC | 红色 | 暂停：立即暂停或大幅降低出价 |
| **? - 未知** | 无点击数据 | 灰色 | 观察：保持运行积累数据 |

## 5. 品牌级实际利润追踪（如有YP平台报表）

### Step 4.6: 对齐时间范围

- Google Ads报告时间范围：[日期A] 到 [日期B]
- YP平台报表时间范围：[日期A] 到 [日期B]
- YP数据通常实时或T+1，无Amazon Associates的24小时Cookie延迟问题

### Step 4.7: 计算品牌级实际利润

```markdown
### 品牌级实际利润分析

| 指标 | 数值 |
|------|------|
| 品牌名 | [品牌名] |
| Google Ads总费用 | ¥[X] (~$[X]) |
| Google Ads总点击 | [N] |
| Google Ads平均CPC | $[X] |
| YP品牌总Clicks | [N] |
| YP品牌Purchases | [N] |
| YP品牌Commission | $[X] |
| **品牌实际EPC** | **$[X]** |
| 盈亏平衡CPC | $[X] |
| **品牌ROI** | **[X]%** |

### 漏斗分析

| 漏斗阶段 | 数值 | 转化率 |
|----------|------|--------|
| YP Clicks | [N] | — |
| Detail Page Views | [N] | [X]% (DPV/Clicks) |
| Add to Carts | [N] | [X]% (ATC/Clicks) |
| Purchases | [N] | [X]% (Purchase/Clicks) |

**关键判断**：
- 如果 Add to Carts > 0 但 Purchases = 0 → 用户在加购后放弃，可能原因：价格敏感、运费、结账流程问题
- 如果 Detail Page Views = 0 → 点击后未到达产品页，检查联盟链接是否正确
- 如果 Commission = 0 但 Amount > 0 → 可能是退货或订单未确认
```

### Step 4.8: 预估转化率校准（品牌级）

```
品牌实际转化率 = YP Purchases ÷ YP Clicks

如果品牌实际转化率 < 预估转化率 × 0.5：
  → 检查Amazon产品页面（评论、评分、库存、价格）
  → 检查广告文案与产品页一致性（是否存在点击后落差）
  → 检查YP联盟链接是否正确（是否指向正确产品页）
  → 考虑降低预估转化率，收紧CPC上限

如果品牌实际转化率 > 预估转化率 × 1.5：
  → 产品表现超预期，可提高CPC上限加码投放

注意：YP Clicks包含所有渠道，如果只做Google Ads：
  实际Google Ads转化率 ≈ YP Purchases ÷ Google Ads Clicks（偏乐观的估计）
```
